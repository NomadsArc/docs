How to display **Promo Discount Banners** on Checkout using the **Notification -\> Banner** option? A Step by Step Guide.

_Promotional Banner Tutorial_

## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![28fda48321 857d2699c5bfb8380988fc759206c8e7](/images/28fda48321-857d2699c5bfb8380988fc759206c8e7.png)

2. Select the **Notifications** category.

![86aed9c810 Ee31babf7c7be101c519f811f5186ce6](/images/86aed9c810-ee31babf7c7be101c519f811f5186ce6.png)

3. Select **Banner** notifications type from the options for displaying the banner on the checkout.

![F39fbea74b 67360cd7ba369aba68d318f01473e02d](/images/f39fbea74b-67360cd7ba369aba68d318f01473e02d.png)

4. Select **Dynamic Render** option from the given placement options.

![Fca81e05df F83e5ffe68bbab6c4b5983ecf5def88f](/images/fca81e05df-f83e5ffe68bbab6c4b5983ecf5def88f.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![5c3b623b17 0d3d5fde5400eb361c16754494988e8b](/images/5c3b623b17-0d3d5fde5400eb361c16754494988e8b.png)

6. Adjust the **type** and **title** for the banner customization block.

![E12b92b439 19d3d068311d551b79e1d4a58915dbdc](/images/e12b92b439-19d3d068311d551b79e1d4a58915dbdc.png)

7. Change the **status** to active to enable the customization block.

![9513611837 38d9a12d25119218234183681810c3f8](/images/9513611837-38d9a12d25119218234183681810c3f8.png)

8. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![B6911049cf D7e0ff341944ce78e8a86644cbc1826f](/images/b6911049cf-d7e0ff341944ce78e8a86644cbc1826f.png)

9. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![18b4b6f378 D243c3593f149667f222ba6a14919486](/images/18b4b6f378-d243c3593f149667f222ba6a14919486.png)

11. Click on the **Save** button to save the changes.

![Bf2cabfba5 8cc209ef1070e05c764af29cbf21a568](/images/bf2cabfba5-8cc209ef1070e05c764af29cbf21a568.png)

12. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![Ebe8a89b62 Efe2e1f4c95f17e55119e3be2418f490](/images/ebe8a89b62-efe2e1f4c95f17e55119e3be2418f490.png)

13. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![E989097815 137901f4c5dacaf188c5d110e4c10acb](/images/e989097815-137901f4c5dacaf188c5d110e4c10acb.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

14. Select the **Checkout - Instruction Banner** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![9ef028fb20 D33f68865079f82d4fc3284ab060e49a](/images/9ef028fb20-d33f68865079f82d4fc3284ab060e49a.png)

15. Click on the **Save** button to publish the changes live for the customers.

![De49128f10 534f1dfe24ab57cf3df9b4b546334931](/images/de49128f10-534f1dfe24ab57cf3df9b4b546334931.png)
