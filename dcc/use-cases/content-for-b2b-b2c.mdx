How to display different image content for **B2B** and **B2C** customers on the Checkout page using the **Notifications -\> Content** option.

_Content for B2B / B2C Tutorial_

## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![](/assets/dynamic-checkout-customizer/28fda48321.png)

2. Select the **Notifications** category.

![](/assets/dynamic-checkout-customizer/86aed9c810.png)

3. Select **Content**, a notifications type from the options for displaying the image or text on the checkout.

![](/assets/dynamic-checkout-customizer/328784f3c3.png)

4. Select **Dynamic Render** option from the given placement options.

![](/assets/dynamic-checkout-customizer/fca81e05df.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![](/assets/dynamic-checkout-customizer/ce9058af5a.png)

6. For displaying content only for B2B customers, add a condition of **customer type** and select the condition value **B2B**.

![](/assets/dynamic-checkout-customizer/8e88290188.png)

7. Adjust the **content settings** for the content customization block which includes **image source** link and **redirect link** for the trust pilot reviews.

![](/assets/dynamic-checkout-customizer/3ca98b8bc6.png)

8. Change the **status** to active to enable the customization block.

![](/assets/dynamic-checkout-customizer/dfdf8a69ed.png)

9. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![](/assets/dynamic-checkout-customizer/065de50336.png)

10. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![](/assets/dynamic-checkout-customizer/a76aa11836.png)

11. Click on the **Save** button to save the changes.

![](/assets/dynamic-checkout-customizer/868a548598.png)

12. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/assets/dynamic-checkout-customizer/ebe8a89b62.png)

13. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/assets/dynamic-checkout-customizer/e989097815.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

14. Select the **Checkout - Image Content** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![](/assets/dynamic-checkout-customizer/6f53fdc188.png)

15. Click on the **Save** button to publish the changes live for the customers.

![](/assets/dynamic-checkout-customizer/70e849ab70.png)

[Gimaguas](https://gimaguas.com/) is using **Notifications -\> Content** customization for displaying Images and improving their Content Marketing Strategy.
