How to **Add Notes** for each cart item on the Checkout using the **Cart Item Details -\> Add Notes** option.

<iframe src="https://www.loom.com/embed/7d714e0ee3224058b51b15d39a034772" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen width="100%" height="480" />

## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![28fda48321 857d2699c5bfb8380988fc759206c8e7 1](/images/28fda48321-857d2699c5bfb8380988fc759206c8e7-1.png)

2. Select the **Cart Item Details** category.

![05437cb8d0 B6b75f30e400f6cd257073a43c965390](/images/05437cb8d0-b6b75f30e400f6cd257073a43c965390.png)

3. Select **Add Notes**, a cart line details type from the options for displaying the quantity selector on the checkout for each cart item.

![Bbab37f999 C703d205e335c20e7dee019f59a3e4fa](/images/bbab37f999-c703d205e335c20e7dee019f59a3e4fa.png)

4. Select **Render After Each Cart Item** option from the given placement options.

![08417d9e12 D7737ceaa15234293f90ea7ab5289fbc](/images/08417d9e12-d7737ceaa15234293f90ea7ab5289fbc.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![2c3411b1b7 2c89fb17d3a49d935096ad59dcdfc9d3](/images/2c3411b1b7-2c89fb17d3a49d935096ad59dcdfc9d3.png)

6. Adjust the **Notes Settings** for the Edit Cart Items customization block which includes **link title**, **notes title** and **notes key** for each cart line item note.

![435f91e039 41b809ee0139f21d0a2e709996f58fe4](/images/435f91e039-41b809ee0139f21d0a2e709996f58fe4.png)

7. Change the **status** to active to enable the customization block.

![D59f66b4ab Da0686f0c32a7a6add2f988283e8fa87](/images/d59f66b4ab-da0686f0c32a7a6add2f988283e8fa87.png)

8. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![Aac0507c4f 55e256f03dc7feb36cfeb1c04b555b7d](/images/aac0507c4f-55e256f03dc7feb36cfeb1c04b555b7d.png)

9. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![Fb27c2c828 Eaa6a614a84be7b02612a8bfc9cfdb65](/images/fb27c2c828-eaa6a614a84be7b02612a8bfc9cfdb65.png)

10. Click on the **Save** button to save the changes.

![](/assets/dynamic-checkout-customizer/8404816493.png)

11. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/assets/dynamic-checkout-customizer/ebe8a89b62.png)

12. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/assets/dynamic-checkout-customizer/e989097815.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

13. Select the **Checkout - Edit Cart Items** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![](/assets/dynamic-checkout-customizer/a64029a329.png)

14. Click on the **Save** button to publish the changes live for the customers. Referesh the page to view the changes on the checkout page.

![](/assets/dynamic-checkout-customizer/e9082274d2.png)

## Why does it matter?

**Cart Item Details -\> Add Notes** customization for adding custom notes can improve AOV by **10%**.
