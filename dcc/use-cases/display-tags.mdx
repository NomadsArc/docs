## Guide

<Steps>
  <Step title="Create a new customization">
    Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

    ![](/images/28fda48321.png)
  </Step>

  <Step title="Select the Cart Item Details category">
    Choose **Cart Item Details** as the customization category.

    ![](/images/05437cb8d0.png)
  </Step>

  <Step title="Choose the Display Tags type">
    Select **Display Tags** from the available Cart Item Details options to display custom product tags for each cart item during checkout.

    ![](/images/b5653aa476.png)
  </Step>

  <Step title="Choose a placement">
    Select **Render After Each Cart Item** from the available placement options.

    ![](/images/08417d9e12.png)
  </Step>

  <Step title="Configure display conditions">
    Add one or more **conditions** to control when the customization is displayed during checkout.

    ![](/images/391aa0a76c.png)
  </Step>

  <Step title="Configure Display Tags settings">
    Configure the **Display Tags** settings by mapping **Product Tags** to the custom tag values you want to display for each cart item.

    ![](/images/1005b6fadc.png)
  </Step>

  <Step title="Activate the customization">
    Change the **Status** to **Active** to enable the customization.

    ![](/images/6e0f08c8a3.png)
  </Step>

  <Step title="Set the priority">
    Configure the **Priority** (0–100) to determine the execution order when multiple customizations are eligible.

    ![](/images/b72ef23d7c.png)
  </Step>

  <Step title="Copy the Placement ID">
    Open the **Placement** tab and copy the generated **Placement ID**. You'll use this in the Shopify Checkout Editor.

    ![](/images/55582f77e9.png)
  </Step>

  <Step title="Save the customization">
    Click **Save** to store your configuration.

    ![](/images/e51f7df7f1.png)
  </Step>

  <Step title="Open the Checkout Editor">
    Go to your Shopify **Settings → Checkout**, select your checkout profile, and click **Customize**.

    ![](/images/ebe8a89b62.png)
  </Step>

  <Step title="Add the app block">
    In the Checkout Editor, click **Add app block**.

    ![](/images/e989097815.png)

    <Note>
      Make sure the app block is placed in the same checkout placement you selected in Dynamic Checkout Customizer.
    </Note>
  </Step>

  <Step title="Configure the app block">
    Select the **Checkout – Edit Cart Items** app block, drag it to the desired location, and paste the copied **Placement ID**.

    ![](/images/c7d92b805d.png)
  </Step>

  <Step title="Publish your changes">
    Click **Save** in the Checkout Editor to publish the customization. Refresh your checkout page to verify that the Display Tags customization is active.

    ![](/images/52a488ae53.png)
  </Step>
</Steps>

<Note>
  **Why does it matter?**

  [BoutiquesByTheBox](https://boutiquebythebox.com) uses the **Cart Item Details → Display Tags** customization to highlight important product information during checkout, creating a more informative shopping experience that can help increase average order value (AOV).
</Note>
