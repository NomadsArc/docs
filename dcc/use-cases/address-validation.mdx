How to validate the **Address lines** using Character limit on the checkout page?

<iframe src="https://drive.google.com/file/d/1VLz73JBkACvzaKPQkVLnkbmrjHWvVpMu/preview" width="640" height="480" allow="autoplay" />

_Address Validation_

## Guide

1. Go to [Settings](https://admin.shopify.com/apps/dynamic-checkout-customizer/settings) page and click [Address Validation](https://admin.shopify.com/apps/dynamic-checkout-customizer/settings) tab.

![4118efee6e Fd645ea19a70246688a09f3ba26885dd](/images/4118efee6e-fd645ea19a70246688a09f3ba26885dd.png)

2. Add a new condition source **Address Line 1** from the Character Limit option.

![4c68f0b559 86865e4a728b954f3c061579db551bae](/images/4c68f0b559-86865e4a728b954f3c061579db551bae.png)

3. **Save** and **Enable** the settings.

![](/assets/dynamic-checkout-customizer/17927f5176.png)

:::info

Incase of custom Regex, please contact our [support](https://calendly.com/san-nomads/30-min-meeting?back=1&month=2024-03) team.

:::

4. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/assets/dynamic-checkout-customizer/ebe8a89b62.png)

5. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/assets/dynamic-checkout-customizer/e989097815.png)

6. Select the **Checkout - Address Validation** app block for Address Validation.

![](/assets/dynamic-checkout-customizer/2a0957ef7c.png)

7. Click on the **Save** button to publish the changes live for the customers. Refresh the page to view the changes on the checkout page.

![](/assets/dynamic-checkout-customizer/8d48b3e7fb.png)

## Why does it matter?

**Address Validation** customization for validation of Address lines can reduce the Shipping and Logistics cost by **10%**.
