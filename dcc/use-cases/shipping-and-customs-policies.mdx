## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![](/images/28fda48321.png)

2. Select the **Notifications** category.

![](/images/86aed9c810.png)

3. Select **Content**, a notifications type from the options for displaying the image or text on the checkout.

![](/images/328784f3c3.png)

4. Select **Dynamic Render** option from the given placement options.

![](/images/fca81e05df.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![](/images/ce9058af5a.png)

6. Adjust the **content settings** for the content customization block which includes **text**, **text** **appearance**, **text size** and **text alignment**.

![](/images/188a845be5.png)

7. Change the **status** to active to enable the customization block.

![](/images/edeb7e6ee7.png)

8. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![](/images/ef15dfc6e5.png)

9. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![](/images/a76aa11836.png)

10. Click on the **Save** button to save the changes.

![](/images/868a548598.png)

11. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/images/ebe8a89b62.png)

12. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/images/e989097815.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

13. Select the **Checkout - Image Content** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![](/images/6f53fdc188.png)

14. Click on the **Save** button to publish the changes live for the customers.

![](/images/bc45a84e77.png)

## Why does it matter?

[Gimaguas](https://gimaguas.com/) is using **Notifications -> Content** customization for displaying Shipping or Customs policies and improved their Brand Reputation by their 5%.
