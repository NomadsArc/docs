How to display **Survey Form** for the customers on the Checkout using the **Notifications -\> Form** option.

<iframe src="https://drive.google.com/file/d/1qjim-C2V6yShMHh57Hf9pkQ5qo1Vyf2n/preview" width="640" height="480" allow="autoplay" />

## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![](/assets/dynamic-checkout-customizer/28fda48321.png)

2. Select the **Notifications** category.

![](/assets/dynamic-checkout-customizer/86aed9c810.png)

3. Select **Form**, a notifications type from the options for displaying the form elements on the checkout.

![](/assets/dynamic-checkout-customizer/b390a3a55b.png)

4. Select **Dynamic Render** option from the given placement options.

![](/assets/dynamic-checkout-customizer/fca81e05df.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![](/assets/dynamic-checkout-customizer/c601b5f168.png)

6. Adjust the **Form settings** for the form customization block which includes element type (**OptionList**), **label** for each form element, **key** for saving the value in the Order details and **block checkout** to reject checkout if the field is empty. You can add multiple options for the Option List.

![](/assets/dynamic-checkout-customizer/5656401da0.png)

7. Change the **status** to active to enable the customization block.

![](/assets/dynamic-checkout-customizer/81e27e6113.png)

8. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![](/assets/dynamic-checkout-customizer/5656401da0.png)

9. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![](/assets/dynamic-checkout-customizer/2e4d6d963f.png)

10. Click on the **Save** button to save the changes.

![](/assets/dynamic-checkout-customizer/ed7c23bdf9.png)

11. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/assets/dynamic-checkout-customizer/ebe8a89b62.png)

12. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/assets/dynamic-checkout-customizer/e989097815.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

13. Select the **Checkout - Form Elements** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![](/assets/dynamic-checkout-customizer/1e4bd03a7a.png)

14. Click on the **Save** button to publish the changes live for the customers.

![](/assets/dynamic-checkout-customizer/7435e6999d.png)
