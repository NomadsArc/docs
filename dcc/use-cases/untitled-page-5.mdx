How to display a **Quantity Selector** for each cart item on the Checkout using the **Cart Item Details -\> Adjust Quantity** option.

<iframe src="https://www.loom.com/embed/450773eaed82428d8c238a73b8ba3bbe" frameborder="0" webkitallowfullscreen mozallowfullscreen allowfullscreen width="100%" height="480" />

## Guide

1. Create a new [customization](https://admin.shopify.com/apps/dynamic-checkout-customizer/customizations).

![](/assets/dynamic-checkout-customizer/28fda48321.png)

2. Select the **Cart Item Details** category.

![](/assets/dynamic-checkout-customizer/05437cb8d0.png)

3. Select **Adjust Quantity**, a cart line details type from the options for displaying the quantity selector on the checkout for each cart item.

![](/assets/dynamic-checkout-customizer/e41420a3a0.png)

4. Select **Render After Each Cart Item** option from the given placement options.

![](/assets/dynamic-checkout-customizer/08417d9e12.png)

5. Add multiple **conditions** to handle the visibility of the customization on the checkout.

![](/assets/dynamic-checkout-customizer/c601b5f168.png)

6. Adjust the **Quantity Selector** settings for the Edit Cart Items customization block which includes **link title** for each cart line item.

![](/assets/dynamic-checkout-customizer/22a04eb26a.png)

7. Change the **status** to active to enable the customization block.

![](/assets/dynamic-checkout-customizer/cff54a0030.png)

8. Adjust the **priority** between 0 to 100 to cover multiple use cases for the customizations.

![](/assets/dynamic-checkout-customizer/7d0a78dd6f.png)

9. In the **placement** tab, copy the placement ID which will be used on the checkout page.

![](/assets/dynamic-checkout-customizer/3b93bc2024.png)

10. Click on the **Save** button to save the changes.

![](/assets/dynamic-checkout-customizer/c4bd2bd678.png)

11. Go to store [Settings](https://admin.shopify.com/settings), click Checkout and then select a checkout profile and click on **customize** button to start the customization process.

![](/assets/dynamic-checkout-customizer/ebe8a89b62.png)

12. In the [Checkout editor](https://admin.shopify.com/settings/checkout/editor), click on the **Add app block** button located at the bottom left corner of the page.

![](/assets/dynamic-checkout-customizer/e989097815.png)

:::info

Make sure you that you have placed the app block at the same placement as selected in the app.

:::

13. Select the **Checkout - Image Content** app block, drag and drop the app block at the desired location and paste the placement ID to view the customization block.

![](/assets/dynamic-checkout-customizer/1e4bd03a7a.png)

14. Click on the **Save** button to publish the changes live for the customers.

![](/assets/dynamic-checkout-customizer/7545f49df4.png)

## Why does it matter?

**Cart Item Details -\> Adjust Quantity** customization for displaying Quantity Selector can improve AOV by **10%**.
